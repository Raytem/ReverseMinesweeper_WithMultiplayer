# Reverse minesweeper game (with diamonds and multiplayer)

## Project setup

```bash
# client
$ cd client
$ npm install

# server
$ cd client
$ npm install
```

## Run the project

```bash
# client
$ npm run serve

# server
$ npm run start
```


## Play game
**To play this game** from different users you should open app in `different browsers` or in `incognito-mode`
